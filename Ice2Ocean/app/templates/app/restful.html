Description of URI templates for the API

Template (start each template string with: http://ice2ocean-api.azurewebsites.net/api/) Description Format Description
RASTERS
get-raster?param={var1}&year={var2}&month={var3}&day={var4}
  • {var1} = 'prec' or 'roff'
  • {var2} = 4 digit year
  • {var3} = month
  • {var4} = day
png Returns .png of requested SnowModel output
VECTORS
get-vector?param={var1}&year={var2}&month={var3}&day={var4}
  • {var1} = 'prec' or 'roff'
  • {var2} = 4 digit year
  • {var3} = month
  • {var4} = day
png Returns .png of requested SnowModel output
TIMESERIES
get-timeseries?table=GRACE&region={var1}&version={var2}
  • {var1} = GRACE region
  • {var2} = version number
csv file date, SUM(mass) (gigatons, summed over the region)
get-timeseries?table=GRACE&mascon={var1}&version={var2}
  • {var1} = mascon number
  • {var2} = version number
csv file date, mass (gigatons)
get-timeseries?table=streamgages&location={var1}
  • {var1} = stream location (in quotes)
 csv file date, gaugeid, discharge
get-timeseries?table=pointbalances&glacier={var1}
  • {var1} = glacier name (in quotes)
csv file start date, end date, stake, elevation (m), balance (m water equivalent), lat/long (WKT format)
